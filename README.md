# Senet-Takip
## c++ diliyle yazılmış senet takip programıdır


## Ekran Görüntüleri ve Programın Açıklanması
### Program çalışmaya başladığında ilk önce karşımıza "senet işlemi" mi yoksa "müşteri işlemi" mi yapmak istediğimizi soran bir menü çıkar. Müşteri eklemek için müşteri işlemlerini seçiyoruz.
![ekle](https://github.com/serifegnll/Senet-Takip/blob/main/senet_ss/kayit.png)
### "Kaydedilsin mi?" diye sorar; hayır (h) cevabını alırsa menüye geri döner, evet (e) cevabını alırsa başka kayıt yapılıp yapılmayacağını sorar. Cevaba göre menüye ya da kayıt ekleme ekranına yönlendirir.
![ekle](https://github.com/serifegnll/Senet-Takip/blob/main/senet_ss/liste.png)
### Müşteri listeleme seçeneğinden eklediğimiz müşterinin bilgilerini görebiliriz.
### Eklediğimiz müşterinin adına senet eklemek için en baştaki menüden senet işlemlerini seçiyoruz. Senet ekle seçeneğini seçiyoruz.
![ekle](https://github.com/serifegnll/Senet-Takip/blob/main/senet_ss/eklemeekrani.png)
### Müşteri numarasına göre ekleme yapılır.
![ekle](https://github.com/serifegnll/Senet-Takip/blob/main/senet_ss/senetekle.png)
### Girilen müşteri numarasına sahip müşterinin bilgileri gösterilir ve bu kişi için senet eklemek istemediğimizi sorar.
![ekle](https://github.com/serifegnll/Senet-Takip/blob/main/senet_ss/eklensinmi.png)
### Senet bilgileri girilir (açıklama, toplam tutar, ödenen tutar)
